# Info
This is a tweak that lets you add custom background images to your dm rows in instagram. Custom backgrounds are "per-user" and they persist after closing the app. The tweak works without a jailbreak. The appropriate resolution for the images is `1290x216`

Tested on `Instagram v407.0.0` `iPhone 14 pro max` `iOS 26.0`

# Building
`make clean package` via Theos (www.theos.dev)
